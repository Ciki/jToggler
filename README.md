jToggler - jQuery Toggler Plugin
=========================

Features
--------
Allows for dynamic creation of toggling element with bunch of settings.
Documentation located inside jToggler.js as jsdoc.

Usage
-----
`$('.toggler').jToggler({ handle: null });`

Requirements
------------
  - jQuery
